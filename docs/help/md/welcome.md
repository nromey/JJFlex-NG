# Welcome to JJFlexRadio

JJFlexRadio is a Windows desktop application for controlling FlexRadio FLEX-6000 and FLEX-8000 series transceivers. It was built from the ground up to be fully accessible with screen readers like NVDA and JAWS.

Whether you're working a pileup on 20 meters or just ragchewing on 80, JJFlexRadio gives you complete keyboard control of your Flex radio with clear audio feedback at every step.

## Quick Start

If you're new to JJFlexRadio, start here:

- **Getting Started** — Install the app, connect to your radio for the first time
- **Screen Reader Setup** — Make sure NVDA or JAWS is configured for the best experience
- **Keyboard Reference** — The complete list of every hotkey (this is the page most folks bookmark)

## What's Here

This help file covers everything in JJFlexRadio:

- **Radio Control** — Tuning, band jumping, mode switching, filters and DSP
- **Audio & Transmission** — PTT safety, audio workshop, meter tones, earcons, TX sculpting
- **Leader Key** — Quick DSP toggles with Ctrl+J
- **Logging** — QSO logging, callbook lookup
- **Settings** — Profiles, preferences, and customization
- **Troubleshooting** — Connection issues, SmartLink, audio problems

## Getting Help Quickly

Press `Ctrl+/` at any time to open the Command Finder. It lets you search every command in JJFlexRadio by name. Just start typing what you want to do.

Press `F1` from anywhere to open this help file. If you're focused on a specific part of the app (like the filters panel or the log), F1 will jump straight to the relevant help page.
