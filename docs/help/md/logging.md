# Quick Log and QSO Grid

JJ Flexible Radio Access includes a built-in logging system for recording your QSOs (contacts).

## Logging Mode

When you Tab into the logging pane, you enter Logging mode. Your hotkeys change to logging-specific shortcuts while you are in Logging mode. Tab out of the logging pane to return to Radio mode and your normal radio hotkeys.

## Quick Log Hotkeys

These keys work when the logging pane has keyboard focus:

| Key | Action |
|-----|--------|
| Alt+C | Jump to the Call field |
| Alt+T | Jump to the His RST field |
| Alt+R | Jump to the My RST field |
| Alt+N | Jump to the Name / Handle field |
| Alt+Q | Jump to the QTH field |
| Alt+S | Jump to the State field |
| Alt+G | Jump to the Grid field |
| Alt+E | Jump to the Comments field |
| Alt+D | Set the date and time to now |

## Working a QSO

A typical logging workflow looks like this:

1. Tab into the logging pane — or press `Ctrl+N` to start a new entry right away.
2. Press `Alt+C` and type the other station's callsign.
3. Press `Alt+T` and enter the RST you gave them.
4. Press `Alt+R` and enter the RST they gave you.
5. Fill in any other fields you want to capture (name, QTH, grid, comments).
6. Press `Ctrl+W` to save the QSO to the log.

## Callbook Lookup

After you enter a callsign, JJ Flexible Radio Access can look it up in online callbook databases to automatically fill in the name, QTH, and grid square fields. See the Callbook Lookup help page for details on setting up your callbook service.

## Managing Your Log

| Key | Action |
|-----|--------|
| Ctrl+N | Start a new log entry |
| Ctrl+W | Save and finalise the current entry |
| Ctrl+Shift+F | Search your log |
| Ctrl+Shift+T | View log statistics |
| F6 | Switch between log panes |
| Ctrl+Shift+N | Open the Log Characteristics dialog |
| Ctrl+Alt+L | Open the full log entry form |
