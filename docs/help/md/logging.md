# Quick Log and QSO Grid

JJFlexRadio includes a built-in logging system for recording your QSOs (contacts).

## Logging Mode

When you Tab into the logging pane, you enter Logging mode. The hotkeys change to logging-specific shortcuts. Tab out of the logging pane to return to Radio mode.

## Quick Log Hotkeys

These keys work when the logging pane has focus:

| Key | Action |
|-----|--------|
| Alt+C | Jump to Call field |
| Alt+T | Jump to His RST field |
| Alt+R | Jump to My RST field |
| Alt+N | Jump to Name/Handle field |
| Alt+Q | Jump to QTH field |
| Alt+S | Jump to State field |
| Alt+G | Jump to Grid field |
| Alt+E | Jump to Comments field |
| Alt+D | Set date/time to now |

## Working a QSO

A typical logging workflow:

1. Tab into the logging pane (or press `Ctrl+N` for a new entry).
2. Press `Alt+C` and type the other station's callsign.
3. Press `Alt+T` and enter the RST you gave them.
4. Press `Alt+R` and enter the RST they gave you.
5. Fill in any other fields as desired (name, QTH, grid, comments).
6. Press `Ctrl+W` to save the QSO.

## Callbook Lookup

After entering a callsign, JJFlexRadio can look it up in online callbook databases to auto-fill name, QTH, and grid. See the Callbook Lookup page.

## Managing Your Log

| Key | Action |
|-----|--------|
| Ctrl+N | Start a new log entry |
| Ctrl+W | Save/finalize the current entry |
| Ctrl+Shift+F | Search your log |
| Ctrl+Shift+T | View log statistics |
| F6 | Switch between log panes |
| Ctrl+Shift+N | Log characteristics dialog |
| Ctrl+Alt+L | Open the full log entry form |
