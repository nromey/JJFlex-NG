# Callbook Lookup

JJFlexRadio can look up callsigns in online databases to auto-fill contact details like name, QTH, and grid square.

## How to Use It

Press `Ctrl+L` from anywhere to open the station lookup dialog and type a callsign. Or, when logging a QSO, enter the callsign in the Call field and JJFlexRadio can look it up automatically.

## Supported Databases

JJFlexRadio supports callbook lookups through services like QRZ.com and HamQTH. You can configure which service to use in the Settings dialog.

## What Gets Filled In

A successful lookup can fill in:

- **Name** — The operator's name
- **QTH** — City and state/country
- **Grid** — Maidenhead grid square
- **State/Province**

Not all fields are available for every callsign — it depends on what the operator has registered with the callbook service.

**Tip:** Callbook lookups require an internet connection. If you're operating without internet (local network only), lookups won't be available.
