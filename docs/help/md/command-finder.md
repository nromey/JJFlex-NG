# Command Finder

The Command Finder is a searchable list of every keyboard command in JJ Flexible Radio Access. It is the fastest way to discover commands you did not know existed, and to remind yourself of a shortcut you have forgotten.

## Opening Command Finder

Press `Ctrl+/` from anywhere in the application to open the Command Finder dialog.

## How It Works

The Command Finder dialog has two controls at the top:

1. **Category filter** — a combo box that lets you narrow the list to a specific category, such as Tuning, DSP, Audio, or Slices. Select "All" to see every command across every category.
2. **Search box** — type a word or phrase to filter the list further within the currently selected category. For example, selecting the DSP category and typing "noise" shows just the noise-related DSP commands.

Below those controls is the command list itself. Each entry shows the keyboard shortcut and a short description of what the command does.

## Tips

- Start with "All" in the category filter if you are just browsing for commands you might not know about. Switch to a specific category once you have a rough idea of what you are looking for.
- The search always works within whatever category is currently selected, so if you are not finding something you know exists, make sure the category filter is set to "All."
- Command Finder is a great way to learn the application — spend a few minutes browsing through the categories, and you will likely find commands you did not know were there.
