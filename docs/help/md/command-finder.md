# Command Finder

The Command Finder is a searchable list of every keyboard command in JJ Flexible Radio Access. It's the fastest way to discover commands you didn't know existed or to remind yourself of a shortcut you've forgotten.

## Opening Command Finder

Press `Ctrl+Shift+H`, or go to the Help menu and select Command Finder.

## How It Works

The Command Finder dialog has two controls at the top:

1. **Category filter** — a combo box that lets you narrow the list to a specific category like Tuning, DSP, Audio, Slices, and more. Select "All" to see every command.
2. **Search box** — type a word or phrase to filter the list further within the selected category. For example, selecting the DSP category and typing "noise" shows just the noise-related DSP commands.

Below those controls is the command list. Each entry shows the keyboard shortcut and a description of what it does.

## Tips

- Start with "All" in the category filter if you're just browsing. Switch to a specific category when you know roughly what you're looking for.
- The search works within whatever category is selected, so if you're not finding something, make sure the category filter is set to "All."
- This is a great way to learn the app — spend a few minutes browsing through the categories and you'll probably find commands you didn't know about.
