# About Dialog

The About dialog shows you version information, system details, and credits for JJ Flexible Radio Access.

## Opening the About Dialog

Go to Help > About, or use whatever shortcut your screen reader provides to navigate the Help menu.

## Tabs

The dialog has three tabs:

### About

Shows the application name (JJ Flexible Radio Access), the current version number, and a tribute to Jim Shaffer, who created the original JJFlexRadio program.

### System Info

Displays technical details about your setup: radio model, FlexLib version, .NET version, and other information that's useful for troubleshooting.

There's a **Copy to Clipboard** button here that copies all the system info as text. You'll hear a confirmation when the copy completes. This is handy when reporting a bug or asking for support — just paste it into your email or message.

### Credits

Acknowledgments for the people and projects that made the app possible.

## Closing

Press `Escape` to close the About dialog. Focus returns to the main window.
