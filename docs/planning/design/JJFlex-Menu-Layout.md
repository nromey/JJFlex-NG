# JJ Flex — Proposed Menu Layout (Modern)

Last updated: 2026-01-31
Status: Draft proposal

## Top-level
- **Radio**
  - Connect (Local)
  - Connect (Remote / SmartLink)
  - Manage Accounts
  - Quick Connect (default rig)
  - Disconnect
- **Slice**
  - Select Slice… (list)
  - Next/Previous Slice
  - Slice Properties…
  - TX Slice… / Set TX to Active Slice
  - Enable/Disable Slice
- **Filter**
  - Narrow / Widen
  - Shift Low Edge / Shift High Edge
  - Presets… (CW narrow, SSB wide, contest)
  - Reset to Default
- **Audio**
  - Adjust Rig Audio… (TX/RX levels, ALC guidance)
  - Record / Playback
  - Route Audio / DAX… (decision pending)
- **Waterfall**
  - Open Waterfall Explorer…
  - Sonification Mode…
  - Peek List… (next/prev peak)
  - Marker Earcons… (on/off)
- **Logger** (optional)
  - Logger Mode On/Off
  - Logging actions (when enabled)
- **Tools**
  - Command Finder (F12)
  - Speak Status
  - Status Dialog
  - Hotkey Editor
- **Help**
  - Quick Start
  - Commands / Shortcuts
  - About

## Notes
- Slice menu is the “home” for slice actions (mode, TX, pan, AGC).
- Filter menu is filter-only (do not mix with mode/TX).
- Classic mode may keep legacy menu structure and Fields menu behavior.
